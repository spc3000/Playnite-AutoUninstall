# Playnite-AutoUninstall
This is a Playnite extension that adds the ability to more easily uninstall manually added games to Playnite.
The extension simplifies the uninstall prcoess for manually added games and lets you do it from PLaynite itself. You can uninstall any game you have selected.
It doesn't interefere with Playnite's own Uninstall function, and adds a context menu item called "Auto Uninstall Tool". The options will display the ability to uninstall and a log to see the process in case the uninstallation did not work correctly (i.e. the game wasn't uninstalled).
To install:
1. Download the "b1c1a22c-e642-4fa4-b282-03e314d7f7b1_1_0.pext" file.
2. Once it is downloaded, simply double click and then click yes when prompted to install the add-on.
